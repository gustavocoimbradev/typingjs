# ⌨️ TypingJS

O **TypingJS** é um plugin leve e simples desenvolvido em JavaScript puro para criar efeitos de máquina de escrever (typewriter) dinâmicos em páginas web. 

Com ele, você pode adicionar interatividade e dinamismo ao seu site, simulando a digitação de múltiplos textos com controle total sobre o tempo e o comportamento do efeito.

---

## ✨ Funcionalidades

* **Custom Speed:** Controle o tempo de escrita por letra.
* **Erase Control:** Ajuste a velocidade com que o texto é apagado.
* **Smart Delays:** Defina o tempo de espera antes de iniciar a limpeza do texto.
* **Interval Management:** Controle o intervalo entre diferentes frases.
* **Infinite Loop:** Opção para manter o efeito de digitação e exclusão em ciclo contínuo.
